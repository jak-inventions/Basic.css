Basic.CSS

About

Basic.CSS is a minimalistic CSS stylesheet that aims to provide basic styling to any web page
with just one line of code. This stylesheet provides the absolute bearbones of a presentable
web page, such as font types, colors, and padding, to help you save more time for development.

Setup

To import Basic.CSS, just add the following line of code at the top of your HTML file:

<link rel="stylesheet" href="https://jak.bz/Basic-CSS/basic.css">
