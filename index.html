Basic.CSS

Basic.CSS is an minimalistic, open-source CSS framework that aims to make any website look more presentable with just one line of code

To import Basic.CSS, just add the following line of code to the <head> of your HTML file:

<link rel="stylesheet" href="https://protosapps.com/Basic-CSS/basic.css"/>
